# rose-js
Common Typescript tooling for web projects.

## Installation 
```sh
npm install branogoga/rose-js --save
```
## Usage
### TypeScript
```typescript
import * as Rose from 'rose-js';
console.log(Rose.add(1,2));
```
```sh
Output should be '3'
```
## Test 
```sh
npm run test
```
